Base URL: https://kioju.de/api/api.php

POST Add Link
Parameters
Parameter	Type	Required	Description
action	string	Required	Must be "add"
url	string	Required	Valid HTTP/HTTPS URL to add
title	string	Optional	Custom title (auto-fetched if empty)
is_private	string	Optional	"1" for private (default), "0" for public
capture_description	string	Optional	"1" to capture description (default), "0" to skip
tags	string | array	Optional	Comma-separated list or repeated field to attach tags ("reading,research")
Example Request
curl -X POST https://kioju.de/api/api.php \ -H "X-Api-Key: your_api_key_here" \ -H "Content-Type: application/x-www-form-urlencoded" \ -d "action=add&url=https://example.com&title=Example Site&is_private=0&capture_description=1&tags=reading,research"
 Success Response (HTTP 201)
{ "success": true, "link_id": 123, "tags": [ { "id": 42, "name": "Reading", "slug": "reading" }, { "id": 43, "name": "Research", "slug": "research" } ] }
 Error Responses
// HTTP 400 - Invalid URL { "success": false, "message": "Invalid URL format" } // HTTP 409 - Duplicate Link { "success": false, "message": "Duplicate link detected", "existing_link": { "id": 456, "url": "https://example.com", "title": "Example Site", "created_at": "2025-09-12 10:30:00" } } // HTTP 429 - Rate Limited { "success": false, "message": "Rate limit: 50/50 submissions used. Try again in 35 minutes", "rate_limit": { "current": 50, "limit": 50, "reset_in_minutes": 35 } }
GET List Links
Parameters
Parameter	Type	Required	Description
action	string	Required	Must be "list"
limit	integer	Optional	Number of links to return (default: 20, max: 100)
offset	integer	Optional	Number of links to skip (for pagination)
Example Request
curl -X GET "https://kioju.de/api/api.php?action=list&limit=10&offset=0" \ -H "X-Api-Key: your_api_key_here"
 Success Response (HTTP 200)
{ "success": true, "links": [ { "id": 123, "url": "https://example.com", "title": "Example Site", "description": "This is an example website", "created_at": "2025-09-12 10:30:00", "is_private": false, "is_shadowbanned": false, "tags": [ { "id": 42, "name": "Reading", "slug": "reading" } ] } ] }

GET Check Premium Status
Determine whether the API key belongs to a Premium account. Helpful for native apps that need to enable or disable premium-only UI features.

Parameters
Parameter	Type	Required	Description
action	string	Required	Must be "premium_status"
Example Request
curl -X GET "https://kioju.de/api/api.php?action=premium_status" \ -H "X-Api-Key: your_api_key_here"
 Success Response (HTTP 200)
{ "success": true, "is_premium": true, "plan": "premium", "feature_access": { "list": true, "delete": true, "update": true } }
GET Search Tags
Returns suggested tags to power autocomplete and encourage normalized tagging.

Parameters
Parameter	Type	Required	Description
q	string	Required	Search query (prefix or partial tag name)
limit	integer	Optional	Maximum suggestions to return (default 10, max 30)
Example Request
curl -X GET "https://kioju.de/api/tag_suggestions.php?q=read" \ -H "X-Requested-With: XMLHttpRequest"
 Success Response (HTTP 200)
{ "success": true, "results": [ { "id": 42, "name": "Reading", "slug": "reading", "usage_count": 58 } ] }
